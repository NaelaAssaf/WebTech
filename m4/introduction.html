BIT02 Web Technologies — Introduction

INTRODUCTION

Web technologies play a crucial role in modern-day computing, allowing us to create dynamic webpages that can be accessed and interacted with through web browsers. This course aims to provide an introduction to the fundamental web technologies: HTML, CSS and PHP. By the end of this course, the student will be proficient in creating static HTML webpages styled with CSS and making them dynamic using PHP.

Embrace Practical Learning

This course thrives on hands-on experience. Engaging with exercises and actively tinkering with code forms the most effective method to truly comprehend and master the concepts highlighted throughout this course.

Resources

The primary documentation for this course is hosted on a website accessible at https://mathiasverbeke0.github.io/WebTech. The advantage of using a website as a textbook lies in the fact that the browser can instantly render and format included examples and code snippets.

Unlike other courses, this course does not rely on slides. Instead, the key concepts will be introduced through practical examples during the lessons. These examples, along with solutions for exercises completed during the course, will be made available in a GitHub repository (Still has to be made available).

An in-depth manual for PHP can be found at http://php.net. For an HTML and CSS manual, see http://www.w3schools.com/html and http://www.w3schools.com/css

Contact

If you have any questions or get stuck during the course, please ask questions through the discussions on LEHO. I will monitor the discussions page and provide answers there. For any other inquiries, feel free to contact me at mathias.verbeke@howest.be.
