BIT02 Web Technologies — Getting started

GETTING STARTED

Prior to delving into the world of web technologies, it's essential to ensure that your computer is equiped with the necessary software and tools.
These encompass:

  • A web browser: A modern web browser is essential for viewing and testing web pages (we will use firefox)
  • A text editor: A reliable text editor is crucial for writing and editing code (we will use visual studio code)
  • Hypertext PreProcessor: PHP is a server-side scripting language used in web development

Normally firefox and php are already installed on your virtual machine. If not, they can easily be installed by running:

sudo dnf install firefox php

Press y when prompted Is this ok [y/N]:.

Visual studio code we will install together during the first lecture.
